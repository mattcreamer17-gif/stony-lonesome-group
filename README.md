# Stony Lonesome Group Landing Page

Bespoke prospect-facing landing page for Stony Lonesome Group (booth 1057). The page mirrors SLG's own visual system: black background, white uppercase Avenir headlines with tight letter-spacing, DIN-family body copy, a muted olive accent (#7A8271), and austere typography-led layout with no consumer-VC chrome. It carries the pitch body from section 7 of the fit analysis, closes on a 30-minute call with Matt Creamer, and keeps Eduba branding to a single footer line.
